# Telco Customer Churn - EDA Project

## Project Overview
This project analyzes customer churn in a telecom company using Python and EDA techniques.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis
- Churn distribution
- Contract vs churn
- Tenure vs churn
- Monthly charges analysis

## Conclusion
Customer churn is influenced by contract type, tenure, and pricing.
